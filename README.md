# scala-cli-gradle-spring-findbugs-jacoco-pmd-mockito-cucumber-junit-car-data

## Description
A POC for spring app using junit, findbugs,
jacoco, pmd, mockito, and cucumber framework.

## Tech stack
- scala
- gradle
  - spring
  - junit
  - cucumber
  - mockito
  - jacoco
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
